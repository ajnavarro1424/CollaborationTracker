# Basic setup #

```
#!bash
brew install python3
brew install mongodb
pip3 install virtualenv
mkdir .../data_sharing
cd .../data_sharing
virtualenv -p python3 venv
. ./venv/bin/activate
pip install -r requirements.txt
```
